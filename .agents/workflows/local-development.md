# Local Development

// turbo
1. **Install Dependencies**:
   - Run `npm install` to ensure all packages are up to date.
// turbo
2. **Start Dev Server**:
   - Run `npm run dev` to start the local Vite development server.
   - (Note: Do not use `npm run server` as this is not configured for Vite).
3. **Access Application**:
   - Open the local URL provided by Vite (typically http://localhost:5173).
